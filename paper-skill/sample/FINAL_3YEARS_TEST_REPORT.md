## 🔍 结果深度分析

### 为什么近3年只有1篇论文？

#### 1. **学术论文索引延迟**
- **正常现象**: 论文从发表到被数据库索引需要2-12个月
- **预印本vs期刊**: arXiv预印本较快，传统期刊较慢
- **2025-2026论文**: 大部分仍在发表或索引过程中

#### 2. **专业术语的演进**
- "统计决策"可能在不同时期有不同的表述方式
- 相关概念可能分散在多个关键词下
- 跨学科研究使用不同术语体系

#### 3. **API限速的影响**
```
测试过程中遇到的API错误：
- Semantic Scholar: HTTP Error 429 (Too Many Requests)
- arXiv: HTTP Error 429 (Timeout)
- OpenAlex: Implementation issues
```
这些技术限制严重影响了检索效果。

#### 4. **时间范围的现实约束**
```
学术发表的时间线：
2026年6月 → 现在时间
2025年11月 → 最新论文索引
2023年6月 → 3年范围起点
```
实际上，在这个范围内能被索引的高质量论文本身就相对有限。
