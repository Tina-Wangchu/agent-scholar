# 带注释的脚本说明

这个文件夹包含两个Python脚本的带详细中文注释版本，专为编程新手设计。

## 📚 文件列表

### 1. `paper_search_annotated.py`
**论文检索脚本 - 带详细中文注释**

这个脚本可以搜索学术论文，并按设定的条件过滤。

**主要功能**：
- 🔍 从多个学术数据库检索论文（Semantic Scholar、arXiv、CrossRef）
- ⏰ 按时间范围过滤（如"近3年"、"2020-2023"）
- 📊 按引用量或相关度排序
- 📝 输出多种格式（JSON、Markdown、CSV）

**适合学习的内容**：
- 如何使用Python发送HTTP请求
- 如何解析JSON和XML数据
- 如何处理日期和时间
- 如何使用正则表达式
- 如何组织面向对象的代码

### 2. `generate_report_annotated.py`
**报告生成脚本 - 带详细中文注释**

这个脚本将检索结果生成专业的PDF报告。

**主要功能**：
- 📄 生成PDF格式的学术报告
- 📋 包含封面页、论文列表、数据分析
- 🎨 自动排版和格式化
- 🌏 支持中英文内容

**适合学习的内容**：
- 如何使用reportlab库生成PDF
- 如何创建文档样式（字体、颜色、布局）
- 如何处理文件输入输出
- 如何创建表格和列表
- 如何构建复杂的文档结构

## ❓ 常见问题

### Q: 注释中的符号是什么意思？
**A**: 
- `#` → 单行注释，解释这一行代码的作用
- `""" """` → 文档字符串，说明函数的用途
- `""" → 三引号，可以写多行注释

### Q: `self` 是什么？
**A**: `self` 表示对象自己，就像说"我自己的属性"。

### Q: `-> List[Dict]` 是什么意思？
**A**: 这是类型提示，说明函数返回一个列表，列表中每个元素是字典。

### Q: 为什么有些代码在 `try-except` 中？
**A**: 这是为了错误处理。如果出错不会让程序崩溃，而是给出友好提示。

### Q: `if __name__ == "__main__":` 是什么？
**A**: 这是Python的入口点标记。只有直接运行脚本时才执行这部分代码。
